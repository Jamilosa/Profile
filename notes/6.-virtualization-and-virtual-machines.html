6. Virtualization and Virtual Machines

Sample
Tags: Automation, Cloud Security, DevOps, Incident Response
Virtualization is technology that creates a virtual version of physical infrastructure, such as servers, storage, and networks.

A hypervisor is the abstraction layer that sits between the physical computer and the VM. It distributes resources across VMs and keeps each VM separate to simulate VM as a separate physical computer.

Type 1 - Bare Metal

  • Oracle VM Server

Type 2 - Hosted

  • VirtualBox

Advantage of Virtual Machines

  • Portability - VMs are easily relocatable throughout the network
  • Scalability - resources can be adjusted as needed
  • Testing Environments - easy to deploy and delete
